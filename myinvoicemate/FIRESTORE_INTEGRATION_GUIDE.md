## Firebase Project Setup

### Step 1: Create Firebase Project

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Click **"Add project"**
3. Enter project name: `myinvoismate` (or your preferred name)
4. **Enable Google Analytics**: Optional (recommended for tracking)
5. Click **"Create project"**
6. Wait for project creation to complete

### Step 2: Enable Firestore Database

1. In Firebase Console, navigate to **Build > Firestore Database**
2. Click **"Create database"**
3. **Choose mode**:
   - **Production mode**: Secure (recommended)
   - **Test mode**: Open access (development only)
4. **Select location**: 
   - Choose `asia-southeast1` (Singapore) for Malaysian users
   - Or your preferred region
5. Click **"Enable"**

### Step 3: Register Your Flutter App

#### For Android:
1. In Firebase Console, click **Add app** > **Android**
2. **Android package name**: `com.example.myinvoicemate` (from `android/app/build.gradle`)
3. **App nickname**: MyInvoisMate Android
4. **Debug signing certificate SHA-1**: Optional (for Google Sign-In)
5. Download `google-services.json`

#### For iOS:
1. Click **Add app** > **iOS**
2. **iOS bundle ID**: `com.example.myinvoicemate` (from `ios/Runner/Info.plist`)
3. **App nickname**: MyInvoisMate iOS
4. Download `GoogleService-Info.plist`

#### For Web:
1. Click **Add app** > **Web**
2. **App nickname**: MyInvoisMate Web
3. Copy the Firebase configuration

### Step 5: Implement FirestoreInvoiceService

Check if `lib/backend/invoice/services/firestore_invoice_service.dart` exists. If not, create it:

```dart
import 'package:cloud_firestore/cloud_firestore.dart';
import '../models/invoice_model.dart';
import '../models/invoice_draft.dart';

class FirestoreInvoiceService {
  final FirebaseFirestore _firestore = FirebaseFirestore.instance;

  // Collection references
  CollectionReference get _invoicesCollection => _firestore.collection('invoices');
  CollectionReference get _draftsCollection => _firestore.collection('invoice_drafts');

  /// Save invoice draft to Firestore
  Future<String> saveDraft(InvoiceDraft draft, String userId) async {
    final draftData = draft.toJson();
    draftData['userId'] = userId;
    draftData['createdAt'] = FieldValue.serverTimestamp();
    draftData['updatedAt'] = FieldValue.serverTimestamp();

    final docRef = await _draftsCollection.add(draftData);
    return docRef.id;
  }

  /// Save finalized invoice to Firestore
  Future<void> saveInvoice(Invoice invoice) async {
    final invoiceData = invoice.toJson();
    await _invoicesCollection.doc(invoice.id).set(invoiceData);
  }

  /// Get draft by ID
  Future<InvoiceDraft?> getDraft(String draftId) async {
    final doc = await _draftsCollection.doc(draftId).get();
    if (!doc.exists) return null;
    return InvoiceDraft.fromJson(doc.data() as Map<String, dynamic>);
  }

  /// Get invoice by ID
  Future<Invoice?> getInvoice(String invoiceId) async {
    final doc = await _invoicesCollection.doc(invoiceId).get();
    if (!doc.exists) return null;
    return Invoice.fromJson(doc.data() as Map<String, dynamic>);
  }

  /// Get user's drafts
  Future<List<InvoiceDraft>> getDraftsByUser(String userId) async {
    final querySnapshot = await _draftsCollection
        .where('userId', isEqualTo: userId)
        .orderBy('createdAt', descending: true)
        .get();

    return querySnapshot.docs
        .map((doc) => InvoiceDraft.fromJson(doc.data() as Map<String, dynamic>))
        .toList();
  }

  /// Get user's invoices
  Future<List<Invoice>> getInvoicesByUser(String userId) async {
    final querySnapshot = await _invoicesCollection
        .where('createdBy', isEqualTo: userId)
        .orderBy('createdAt', descending: true)
        .get();

    return querySnapshot.docs
        .map((doc) => Invoice.fromJson(doc.data() as Map<String, dynamic>))
        .toList();
  }

  /// Update invoice compliance status
  Future<void> updateComplianceStatus({
    required String invoiceId,
    required ComplianceStatus status,
    String? myInvoisReferenceId,
  }) async {
    final updateData = <String, dynamic>{
      'complianceStatus': status.toString(),
      'updatedAt': FieldValue.serverTimestamp(),
    };

    if (myInvoisReferenceId != null) {
      updateData['myInvoisReferenceId'] = myInvoisReferenceId;
      updateData['submissionDate'] = FieldValue.serverTimestamp();
    }

    await _invoicesCollection.doc(invoiceId).update(updateData);
  }

  /// Delete draft
  Future<void> deleteDraft(String draftId) async {
    await _draftsCollection.doc(draftId).delete();
  }

  /// Delete invoice
  Future<void> deleteInvoice(String invoiceId) async {
    await _invoicesCollection.doc(invoiceId).delete();
  }
}
```

## Security Rules

### Step 7: Set Up Firestore Security Rules

In Firebase Console, go to **Firestore Database > Rules** and add:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Helper function to check if user is authenticated
    function isAuthenticated() {
      return request.auth != null;
    }
    
    // Helper function to check if user owns the document
    function isOwner(userId) {
      return isAuthenticated() && request.auth.uid == userId;
    }
    
    // Invoice drafts - users can only access their own drafts
    match /invoice_drafts/{draftId} {
      allow read: if isAuthenticated() && 
                     resource.data.userId == request.auth.uid;
      allow create: if isAuthenticated() && 
                      request.resource.data.userId == request.auth.uid;
      allow update, delete: if isAuthenticated() && 
                              resource.data.userId == request.auth.uid;
    }
    
    // Finalized invoices - users can only access their own invoices
    match /invoices/{invoiceId} {
      allow read: if isAuthenticated() && 
                     resource.data.createdBy == request.auth.uid;
      allow create: if isAuthenticated() && 
                      request.resource.data.createdBy == request.auth.uid;
      allow update: if isAuthenticated() && 
                      resource.data.createdBy == request.auth.uid;
      allow delete: if false; // Invoices should not be deleted (compliance)
    }
    
    // Deny all other access
    match /{document=**} {
      allow read, write: if false;
    }
  }
}
```

**Test Mode (Development Only - Not Secure!):**
```javascript
rules_version = '2';
```

## Troubleshooting

### Common Issues

#### Issue 1: "No Firebase App '[DEFAULT]' has been created"
**Solution**: Ensure `Firebase.initializeApp()` is called in `main()` before `runApp()`.

#### Issue 2: "Permission denied" in Firestore
**Solution**: 
- Check Firestore security rules
- Ensure user is authenticated
- Verify userId matches in rules

#### Issue 3: Android build fails with "google-services.json not found"
**Solution**:
- Place `google-services.json` in `android/app/`
- Add to `android/app/build.gradle`: `apply plugin: 'com.google.gms.google-services'`
- Add to `android/build.gradle`: `classpath 'com.google.gms:google-services:4.4.0'`

#### Issue 4: iOS build fails
**Solution**:
- Place `GoogleService-Info.plist` in `ios/Runner/`
- Run `cd ios && pod install`
- Clean build: `flutter clean && flutter pub get`

#### Issue 5: Web configuration not working
**Solution**:
- Ensure Firebase SDK scripts are in `web/index.html`
- Check `firebase_options.dart` is generated
- Verify API keys are correct

#### Issue 6: Data not syncing
**Solution**:
- Check internet connection
- Verify Firestore rules allow write access
- Check console for error messages
- Try `await _firestore.enablePersistence()` for offline support

## Best Practices

### 1. User Authentication
Always use proper user authentication before enabling Firestore:
```dart
// Get current user ID from Firebase Auth
final userId = FirebaseAuth.instance.currentUser?.uid ?? 'anonymous';
```

### 2. Error Handling
Wrap Firestore calls in try-catch:
```dart
try {
  await _firestoreService.saveInvoice(invoice);
} catch (e) {
  if (e is FirebaseException) {
    print('Firebase error: ${e.message}');
  }
}
```

### 3. Offline Support
Enable persistence for offline access:
```dart
await FirebaseFirestore.instance.enablePersistence();
```

### 4. Cost Optimization
- Use `.limit()` for queries
- Cache data locally when possible
- Avoid reading entire collections
- Use indexes for complex queries

### 5. Data Validation
Validate data before saving:
```dart
if (invoice.totalAmount < 0) {
  throw Exception('Invalid invoice amount');
}
```
